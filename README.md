## Prerequisites

- Node.js (v18+)
- npm

## Getting Started

```bash
npm install
npx playwright install
```

## Running Tests

```bash
npx playwright test
```

## Running Tests in UI Mode

```bash
npx playwright test --ui
```

## Viewing HTML Report

```bash
npx playwright show-report
```
